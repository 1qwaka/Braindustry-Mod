# Braindustry-Mod
High Technology mod for Mindustry game.Adds materials,fabrics and new reactor.
How to install?
1)Download .zip of this mod.
2)Launch "mindustry".
3)Click on the "mods" button in start menu.
3)Click "import mod".
5)Find "braindustry-mod-master.zip" and click on him.
6)Click "load".
7)Enjoy.
